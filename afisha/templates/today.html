Сеголня мы научились создавать проект на Django-Templates

1. Активировали venv

2. Установили Django, с поиощью команды pip install Django

3. Создали проект, с помощью команды django-admin startprogect name

4. Создали приложение, с помощью команды pythpn manage.py startapp app

5. В настройках сохранили наше приложение, 'app'

6. Создали папку templates, где будет храниться шаблон для страницы index/

7. Создали view для нашего приложения

8. Создали путь для нашего приложения, path('index/', views.index)

9. Запустили проект с помощью команды python manage.py runserver
